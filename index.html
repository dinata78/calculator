Calculator

Shortcut Keys for Keyboard User

  • Key [ 0-9 ] => Appends number ( 0-9 ) to the calculator's current value
  • Key [ + ] => Appends add operator ( + ) to the calculator's current value
  • Key [ - ] => Appends subtract operator ( - ) to the calculator's current value
  • Key [ * ] => Appends multiply operator ( x ) to the calculator's current value
  • Key [ / ] => Appends divide operator ( / ) to the calculator's current value
  • Key [ . ] => Appends dot symbol ( . ) to the calculator's current value
  • Key [ % ] => Appends percentage symbol ( % ) to the calculator's current value
  • Key [ Delete ] => Clears calculator's current value
  • Key [ Backspace ] => Deletes the last character from the calculator's current value
  • Key [ Control ] => Focus on change theme input
  • Key [ Escape ] => Unfocus change theme input if it has focus
  • Key [ Shift ] => Open the current info page
NOTE: Using the key [ Escape ] here will close the current info page instead.
